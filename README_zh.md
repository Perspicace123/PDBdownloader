**离线 PDB 链接生成器**
一款专为结构生物学研究人员设计的静态网络工具。输入 PDB 编号即可立即生成对应结构文件及相关数据库页面的直接下载链接。

**Read this in other languages: [English](README.md), [中文](README_zh.md)**

**在线试用**
- https://pdbdownloader.netlify.app/

**本地运行**
- 打开“index.html”文件，您即可轻松使用它。

**主要优势**
* 离线运行，响应迅速且稳定，且不存在隐私泄露的风险。
* 可轻松下载分子数据以提高研究效率。

**功能**
*   **即时链接生成**：输入 PDB 编号即可获取完整的直接链接。
*   **多种文件格式**：生成适用于 PDB、mmCIF、XML 文件及其压缩版本的链接。
*   **集成数据库访问**：直接链接至 RCSB PDB、PDBe、PDBsum 和 UniProt 的条目页面以及搜索界面。
*   **隐私优先且离线可用**：完全在浏览器中运行；不会将任何数据发送至任何服务器。
*   **响应式设计**：可在桌面和移动设备上使用。
*   **复制功能**：一键将任何生成的链接复制到剪贴板。
